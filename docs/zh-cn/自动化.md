# 自动化

Project Alice 提供了多项自动化功能，以帮助处理从 Victoria 2 继承下来的一些繁琐工作。

## 作战计划（Battleplanner）

此功能有助于：
- 跨海运送大批军团
- 围攻无人防守的岛屿和领土
- 在选定的省份保持防御部队。

要打开作战计划，请按 `Ctrl-Z` 热键。

### 集团军（Army Groups）

此功能的核心概念是集团军。任何集团军由以下部分组成：
- 附属军团列表。
- 附属海军列表。
- HQ — 总部。这是所有附属空闲军团集结的位置。
- 防御命令列表。集团军会尝试在目标位置保持不超过补给上限的部队。它会尝试使部队类型比例与集团军中的单位比例相似。
- 围攻命令列表。集团军会尽可能围攻这些地点。
- 渡运命令列表。集团军会尝试使用这些省份作为主要港口来运送部队。

与集团军的交互非常直观。

- 要创建集团军，选择一个省份并按下集团军列表上方的 `Create Army Group` 按钮（热键：`Z`）。此操作将创建一个新的集团军，并将所选省份设为总部。
- 所有创建的集团军都列在右侧窗口中。你可以通过点击列表中的项目来选择和取消选择集团军。
- 要添加或移除新的军团或海军，请按下 `Add/Remove units` 按钮。它会将你带到"Add units to battleplanner"场景。选择所需的单位并按下相应的按钮。
