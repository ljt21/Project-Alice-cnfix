# . 军队命令

**作为玩家，**
**我想要控制我的军队，**
**以实现军事目标（战争目标或镇压叛乱）。**

| AC1 | 军队可以被移动到一个省份，攻击那里的任何敌人。 |
| AC2 | 当站在可占领的省份上时，军队会占领它 |

# . 特殊军队命令

**作为玩家，**
**我想要提前规划军队行为，**
**以减少繁琐的微管理。**

**验收标准：**

| AC1 | 默认军队命令是"移动到目的地后停止" |
| AC2 | 新的移动命令会重置军队的特殊军队命令 |
| AC3 | 选择新军队会重置选定的军队命令 |

#  移动并围攻军队命令

**作为玩家，**
**我想要让军队占领沿途的若干省份，**
**以减少繁琐的微管理。**

**验收标准：**

| AC1 | 军队可以被命令"移动并围攻"，意味着在围攻完当前省份前不会移动到路径上的下一个省份 |
| AC2 | "移动并围攻"可以通过命令按钮切换 |
| AC3 | "移动并围攻"可以通过 "V" 热键切换 |

#  战略再部署军队命令

**作为玩家，**
**我想要快速移动不参与战斗的军队，**
**以减少繁琐的微管理。**

**验收标准：**

| AC1 | 军队可以被命令"战略再部署"，提高其速度但将组织度限制在 10% |
| AC2 | "战略再部署"可以通过命令按钮切换 |
| AC3 | "战略再部署"可以通过 "B" 热键切换 |

#  追击并交战军队命令

**作为玩家，**
**我想要派遣我的军队追击并交战敌人，**
**以便在敌人试图逃跑时减少繁琐的微管理。**

**验收标准：**

| AC1 | 军队可以被命令"追击并交战"另一支军队 |
| AC2 | 当被命令"追击并交战"时，军队会追击目标军队并在可能时与之交战 |
| AC3 | "追击并交战"可以通过命令按钮切换 |
| AC4 | "追击并交战"可以通过 "N" 热键切换 |
