# 舰船建造

**作为一国，**
**我想要建造一艘舰船，**
**以便它出现在我的海军中。**

**验收标准：**
| AC1 | 特定类型的舰船由一国在某省份招募 |
| AC2 | 舰船只能在沿海省份招募 |
| AC3 | 只有当该类型对本国可用时，才能招募该类型舰船 |
| AC4 | 当招募所需的所有货物都被购买并由建造消耗后，舰船招募完成 |
| AC5 | 舰船招募不能快于单位建造时间 |
| AC6 | 大型舰船只能在拥有适当海军基地等级的省份招募 |
| AC7 | 当同一省份同时进行多个舰船建造时，它们一次一个地推进 |
