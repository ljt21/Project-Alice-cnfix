## 文档

### 文档生成

目前所有文档都是由 Markdown 文件完成的。网站是通过 doxygen 生成的，这意味着如果愿意，我们也可以使用它。目前我们仅用它来创建网站。我们使用 https://github.com/jothepro/doxygen-awesome-css 进行样式设计，并做了一些小的调整。文档是通过 `docs/generate_docs.sh` shell 脚本生成的。

#### 如何组织文件结构

现在到了重要的部分，

- Markdown 文件应位于 `docs/` 文件夹中
- 侧边栏将与文件结构保持相同的格式
- 如果文件与其所在的文件夹同名，它将被用作该文件夹的"起始页"。否则，它将只显示文件夹中的所有文件
- 文件夹和文件名都不能包含空格
- 文件应以 Markdown 标题开头，这将成为侧边栏和页面中的标题
- [about.md](about.md) 默认是起始页
- 可以通过在第一个标题下方添加 `[TOC]` 来添加目录
