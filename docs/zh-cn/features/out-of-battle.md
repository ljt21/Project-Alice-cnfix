# US13 军团组织度恢复

**作为一国，**
**我希望补充我的军团的组织度，**
**以便它们能在下一场战斗中作战。**

**验收标准：**
| AC1 | 军团的组织度每日恢复 |
| AC2 | 组织度恢复因已满足的军队维护费（军队补给）而降低 |
| AC3 | 国家 `org_regain` 修正增加组织度恢复速率 |
| AC4 | 将军的特质可以增加组织度恢复速率 |
| AC5 | 将军的威望（经验）增加组织度恢复速率 |
| AC6 | 军团的最大组织度为 100% (1.0) |
| AC7 | 未满足的补给不会降低最大组织度，因为这会让半个游戏无法玩 |
| AC8 | 未满足的补给不会阻止组织度恢复，因为这会让半个游戏无法玩 |

# US101 围攻与占领

| AC1 | 堡垒一次性建造，建造成本为一些货物，相当于额外一级工厂。 |
| AC2 | 堡垒每级通过 alice_fort_siege_slowdown 系数（默认为 0.75）降低围攻速度。 |
| AC3 | 堡垒每级通过 state.defines.alice_fort_siege_attrition_per_level（默认为 0.35）增加敌方围攻损耗。 |
| AC4 | 堡垒对省份的胜利点分数有适度影响（每级 +1）。 |
| AC5 | 工兵降低有效堡垒等级。将等级降低 1 级所需的工兵数量取决于围攻军队规模。至少需要 state.defines.engineer_unit_ratio（默认为 0.1）。 |
| AC6 | 侦察单位提高任何省份的占领速度。至少需要 state.defines.recon_unit_ratio（默认为 0.1）。 |
| AC7 | 国家对其自有省份的围攻速度快 25%。 |
| AC8 | 国家对其核心省份的围攻速度快 10%。 |
