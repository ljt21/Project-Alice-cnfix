# 工厂改造（Factory Refit）

玩家可以在生产窗口的工厂列表中将工厂从一种类型改造为另一种类型（向上箭头按钮）。

改造货物成本 = (目标类型的建造成本) - (原类型的建造成本) + (0.1f * 目标类型的建造成本)

将工厂从一种类型改造为另一种类型的条件：
  - 国家已开化
  - 这不是殖民地
  - 已满足建造工厂的技术要求（工厂始终可用，或已研究解锁该建筑的科技）
  - 工厂当前未被升级
  - 或：
	- 执政党允许建造工厂
	- 新旧类型的输入相同或输出相同

最后一个条件意味着，在真正的民主和资本主义经济中，国家代表的政府不能违反私有财产法，强迫资本家改变其工厂的生产线。
