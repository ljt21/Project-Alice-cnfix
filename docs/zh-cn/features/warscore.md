# US5. 战争分数

**作为一国，**
**我希望有一个明确的衡量战争胜负的指标，**
**以便我能强制执行我的要求。**

**验收标准：**
| AC1 | 赢得战斗增加战争分数 |
| AC2 | 被封锁的省份增加战争分数 |
| AC3 | 由一方控制的战争目标增加 ticking（持续累积的）战争分数 |
| AC4 | 将占领省份的分数计入战争中的占领分数。 |
| AC5 | 在较晚开始的战争中，50% 的省份占领分数会计入较早的战争，以防止无限战争。 |
