# US3. 贸易

**作为一国，**
**我希望与我各州关联的市场之间进行货物贸易，**
**以便解决自然的供需失衡。**

**验收标准：**
| AC1 | 市场在州一级创建 |
| AC2 | 市场有与贸易路线管理相关的劳动力成本 |
| AC3 | 市场有与州内贸易相关的劳动力成本 |
| AC4 | 市场通过贸易路线进行出口和进口 |
| AC5 | 海军基地使一个州对贸易路线的吸引力增加 25% |
| AC6 | 移动成本折扣降低了州之间的有效距离，增加了贸易路线出现的吸引力 |
| AC7 | 如果 disallow_naval_trade 国家修正非零，则其各州不能通过海路贸易 |
| AC8 | 如果 disallow_land_trade 国家修正非零，则其各州不能通过陆路贸易 |
| AC9 | 如果双方处于战争状态，则他们之间的贸易暂停（战时禁运） |
| AC10 | 如果一方对另一方设置了禁运，则双方不能彼此贸易 |
| AC11 | 如果势力范围领袖设置了禁运，其势力成员会跟随该禁运。 |
| AC12 | 如果宗主国设置了禁运，其附庸国会跟随该禁运。 |
| AC13 | 商人从原产地与目标地之间的价格差额中抽取一部分 |
| AC14 | 在国际贸易中，同时适用原产地国的出口关税和目标国的进口关税。 |
| AC15 | 强加于某国的不平等条约会取消其出口的出口关税和进口的进口关税。 |
| AC16 | 战时封锁会完全停止该州市场的贸易。 |
| AC17 | 战时占领会完全停止该州市场的贸易。 |
| AC18 | 在运输过程中，根据距离，部分货物会损失。 |
| AC19 | 本地货物和珍贵货物不能进行贸易。 |
| AC20 | 尚未解锁的商品不会被贸易。 |
| AC21 | 贸易量增加会降低运输成本。 |
