# . 军团招募

**作为一国，**
**我想要招募一个军团，**
**以便它出现在我的军队中。**

**验收标准：**
| AC1 | 特定类型的军团由一国在某省份招募 |
| AC2 | 只有当该省份的某个人口允许新招募时，军团才能被招募 |
| AC3 | 只有当该类型对本国可用时，才能招募该类型军团 |
| AC4 | 当招募所需的所有货物都被购买并由建造消耗后，军团招募完成 |
| AC5 | 军团招募不能快于单位建造时间 |
| AC7 | 当同一省份同时进行多个军团建造时，它们同时推进 |
